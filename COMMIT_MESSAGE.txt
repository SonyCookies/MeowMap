feat: add profile screen, menu drawer, and database-backed notification system

This commit implements a comprehensive user profile management system, navigation drawer, and persistent notification system using Supabase database storage.

## New Features

### Profile Management
- Added ProfileScreen.js for viewing and editing user profiles
- Profile editing with validation (display name, bio, location, phone number)
- Avatar upload and preview functionality
- Real-time form validation with error messages
- Profile updates trigger notifications

### Navigation Drawer
- Created MenuDrawer component with slide-in animation from right
- Menu items: Profile, My Sightings, Map View, Community, Statistics, Settings, Help, About, Sign Out
- Consistent animation style with NotificationDrawer
- Clean, organized menu item design

### Notification System (Database-Backed)
- Migrated from AsyncStorage to Supabase database storage
- Created notifications table schema with RLS policies
- Created notificationService.js for database operations
- Notifications persist across app reinstalls and sync across devices
- User-specific notifications with proper security policies

- Notifications are stored per user in Supabase with proper RLS policies
- Profile updates automatically create "Profile Updated" notifications
- Notification badge shows unread count dynamically
- All notification operations (create, read, delete) sync with database
- Menu drawer and notification drawer share consistent animation patterns
- Profile screen includes form validation, avatar upload, and error handling
