refactor(auth): extract components, hooks, and utilities from AuthScreen

This commit refactors the authentication system by extracting reusable
components, custom hooks, and utility functions from AuthScreen.js to
improve code organization, maintainability, and reusability.

## Extracted Components (components/auth/)
- EmailInput: Reusable email input with validation
- PasswordInput: Password input with show/hide toggle
- PasswordStrengthIndicator: Password strength checker with requirements
- RememberMeCheckbox: Remember me checkbox component
- ErrorModal: Reusable error modal component
- SuccessModal: Success message modal component
- TermsModal: Terms and conditions modal
- PrivacyModal: Privacy policy modal

## Custom Hooks (hooks/)
- useBiometricAuth: Handles biometric authentication logic and state
- useLoginAttempts: Manages login attempt tracking and rate limiting

## Utility Functions (utils/)
- emailValidation: Email format validation
- passwordStrength: Password strength checking and requirements
- cooldown: Cooldown time formatting utility
